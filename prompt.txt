Prompt 1:Implement Full Scraper for Entertainment News + Cartoon

Build a complete Playwright-based Python scraper in scraper.py to extract data from https://ekantipur.com.

Requirements:
- Use Playwright sync API (from playwright.sync_api import sync_playwright)
- Do NOT create new files; modify scraper.py only
- Implement a main() function and run everything from there

Task 1: Entertainment News
- Navigate to https://ekantipur.com/entertainment
- Use the DOM structure:
  - Main wrapper: div.category-wrapper
  - Article container: div.category (or article.teaser)
  - Title: h2 a (visible text)
  - Image: div.image img (src or data-src if lazy-loaded)
  - Author: div.author a (if missing, set null)
- Extract top 5 articles
- Each article must include:
  - title (Nepali text exactly as displayed)
  - image_url (full URL)
  - category (e.g., "मनो रञ्जन")
  - author (string or null)

Task 2: Cartoon of the Day
- Navigate to https://ekantipur.com/cartoon
- Locate:
  - Cartoon container: div.cartoon-image
  - Image: div.cartoon-image img (src or data-src)
  - Title: from img alt/title or nearby caption if available
  - Author: cartoonist name if present (span/div class or caption)
- If author is not found, set null

Output Format:
Save results as output.json in exactly this structure:

{
  "entertainment_news": [...],
  "cartoon_of_the_day": {
    "title": "...",
    "image_url": "...",
    "author": "..."
  }
}

Important:
- Ensure all text is UTF-8 (Nepali must remain in Nepali)
- Use ensure_ascii=False when writing JSON
- Handle missing elements safely using try/except or conditional checks
- Use proper waits like page.wait_for_load_state("networkidle") and wait_for_selector
- Use headless=False for debugging
- Do not hardcode any data; everything must be scraped dynamically
- Add clear comments explaining each step
